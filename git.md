# stashing

##  stash changes
```
git stash
```
## *  switch to branch you want to apply changes to, or the branch you want to add another changes

` git switch `
## *  switch back

` git switch `
## *  pop stash

` git stash pop`


# how to delete a branch
### delete branch locally

`git branch -d localBranchName`

### delete branch remotely

`git push origin --delete remoteBranchName`

### refresh branch list

`git fetch -p`
